Hexo教程：使用Hexo写博客

发表于 2019-07-31

新建文档

假设我们新建的文章名为 “hello hexo markdown”，在命令行键入以下命令即可：

$ hexo new “hello hexo markdown”

上述命令的结果是在 ./hexo/source/_posts 路径下新建了一个 hello-hexo-markdown.md 文件。

然后，我们就可以打开编辑器尽情地写作了。
