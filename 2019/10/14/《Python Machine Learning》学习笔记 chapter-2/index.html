Artificial neurons - a brief glimpse into the early history of machine learning

The formal definition of an artificial neuron

Input values: $\boldsymbol x$

Corresponding weights: $\boldsymbol w$

Net Input: $z = w_0x_0 + w_1x_1 + … + w_mx_m = \boldsymbol w^T \boldsymbol x$ , where $w_0 = -\theta$ (bias unit) and $x_0 = -1$

Decision function: $\phi(z)$

The perceptron learning rule

It’s can be summarized by the following steps:

  • Initialize the weights to 0 or small random numbers.
  • For each training sample $\boldsymbol x^{(i)}:$
    • Compute the output value $\hat y$
    • Update the weights
